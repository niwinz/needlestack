=============
ElasticSearch
=============

This documentation covers a ElasticSearch backend of needlestack package. ElasticSearch is a distributed,
RESTful, free/open source search server based on Apache Lucene.

.. note::

    This documentation is independent of other backends available on needlestack. It have posibility to have
    some similitude but it not have any garanty to have same interface.
